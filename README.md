## 📌 Overview

This is a **full-stack portfolio website** built to showcase my projects, skills, and experience as a developer.  
It features a modern UI, smooth animations, and a fully functional contact system powered by a backend API.

---

## 🛠️ Tech Stack

### 🔹 Frontend
- Next.js (React Framework)
- TypeScript
- Tailwind CSS
- Framer Motion (Animations)

### 🔹 Backend
- Spring Boot (Java)
- REST API

### 🔹 Deployment
- Frontend: Vercel
- Backend: Render

---

- 🎨 Modern and responsive UI design
